## 2. issue 글 작성 후 Submit new issue 버튼 클릭
![issue2](https://user-images.githubusercontent.com/54580802/208859315-d5d0d173-73b6-4c1a-8867-528a5a5d2a47.png)

- Labels : 카테고리
- Projects : 프로젝트
